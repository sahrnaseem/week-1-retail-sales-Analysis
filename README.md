Retail Sales Dataset –Week 1 Task

1. Project Title

Retail Sales Dataset Analysis using Microsoft Excel

2. Project Objective

The main objective of this project is to analyze a retail sales dataset using Microsoft Excel. The project focuses on organizing, cleaning, analyzing, and visualizing sales data to gain meaningful business insights.

3. Dataset Information

The dataset contains retail sales records with the following columns:

Transaction ID

Date

Customer ID

Gender

Age

Product Category

Quantity

Price per Unit

Total Amount


4. Data Cleaning

The dataset is cleaned to improve data quality by:

Checking for blank cells.

Identifying duplicate records.

Verifying Quantity and Price per Unit values.

Creating a Calculated Total (Quantity × Price per Unit).

Comparing the Calculated Total with the Total Amount using the IF formula.


5. Excel Functions Used

The following Excel functions were used:

SUM

AVERAGE

MIN

MAX

COUNT

These functions are used to calculate total sales, average sales, total transactions, total quantity sold, and category-wise sales.

6. Sorting and Filtering

Sorting and filtering features are applied to:

Arrange data in ascending and descending order.

Filter records by Gender, Product Category and Date.

Analyze specific groups of data more effectively.


7. Data Analysis

The dataset was analyzed to identify:

Total Sales

Average Sales

Total Number of Transactions

Total Quantity Sold

Sales by Gender

Sales by Product Category

Best-performing Product Category


8. Data Visualization

Charts are created to present the analysis visually, including:

Bar Chart

Column Chart


These charts make it easier to compare product categories and understand sales trends.

9. Key Insights

The analysis helps to:

Identify the highest-selling product category.

Compare sales between different categories.

Understand customer purchasing patterns.

Support better business decision-making.


10. Overall Summary

This project demonstrates the use of Microsoft Excel for analyzing a retail sales dataset. The data was organized, cleaned, and checked for accuracy before analysis. Various Excel functions such as SUM, AVERAGE, COUNT are  used to calculate sales and compare results. Sorting, filtering, and charts are applied to identify sales trends and product performance. The project provides meaningful business insights and improves practical skills in data cleaning, analysis, and data visualization using Excel.
